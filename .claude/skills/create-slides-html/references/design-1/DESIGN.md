# Design System Inspired by Maurelle

> 元資料: `Maurelle.pptx` / 全8スライド / キャンバス **26.66" × 15"**（16:9 の2倍大判）。
> **スケール注記:** ポイント値は2倍キャンバス基準の実測値。標準 1280×720 (13.333"×7.5") で再現する場合はすべての pt とインチ値を **0.5倍** する。本書では原寸（2×）の実測値を記載し、必要箇所に1×換算を併記する。

## 1. Visual Theme & Atmosphere

Maurelle はティール〜ブルー〜インディゴの寒色グラデーション群を主役に据えた、フラットでミニマルな企業プレゼンテーマである。装飾的な影・枠線・角丸を一切持たず、純粋な矩形・正方形・円のジオメトリと大きな余白で構成される。表紙は単色のティール（`#00A09C`）を全面に敷き、その上に一回り小さい同系色の正方形（`#0098A5`）を重ね、白の極太タイトルを配置する「色面の入れ子」が象徴的なモチーフ。コンテンツスライドは白背景に、見出しを `#494949`（ダークグレー）の Montserrat SemiBold、本文を `#999999`（ミディアムグレー）の Lato Light で組み、色は写真ブロック・統計図形・バッジにのみ差し色として使う。フッターには `#F2F2F2` の淡灰バンドをページ番号置き場として中央下部に置く。全体として「彩度の高い寒色を面で使い、文字情報はグレースケールで静かに整える」というコントラスト設計が一貫している。

**Key Characteristics:**

- 寒色グラデーション群（ティール `#00A09C` → ブルー `#1991AB` → インディゴ `#5267A5`）を面・図形の差し色として使用
- 見出し `#494949` / 本文 `#999999` の2段グレースケールでテキスト階層を表現
- 完全フラット（影なし・枠線なし）、角丸なしの鋭いジオメトリ（正方形・矩形・正円）
- 見出しは Montserrat SemiBold、本文は Lato Light の2フォント構成
- 大判キャンバスを活かした広い余白と、3カラム / 3カード型の反復グリッド
- 表紙のみ全面ティール背景＋同系色正方形の入れ子という独立した構造
- `#F2F2F2` の淡灰バンドをフッター（ページ番号領域）として全コンテンツ面に共通配置

## 2. Color Palette & Roles

### Primary — ブランド主色
- **Primary Teal — `#00A09C`** : 表紙の全面背景、第1アクセント、統計数値（例「25%」）、カード1枚目。最頻出のブランド色。

### Accent Colors — アクセント（寒色6段グラデーション）
テーマの accent1〜6 が連続した寒色スペクトラムを成す。実スライドでは主に先頭3色が使われる。
- **Accent 2 — `#0098A5`** : 表紙の入れ子正方形、プログレスバーの塗り、統計数値「50%」、カード2枚目。
- **Accent 3 — `#1991AB`** : 統計数値「70%」、カード3枚目、チャート最大バブル。
- **Accent 4 — `#2B85AE`** : 拡張用（スライド未使用、グラデーション中間）。
- **Accent 5 — `#4175A9`** : 拡張用（スライド未使用）。
- **Accent 6 — `#5267A5`（インディゴ）** : グラデーション終端色。拡張用。

### Dark & Neutral — ダーク・ニュートラル系
- **Heading Ink — `#494949`**（theme tx2/dk2）: 全見出し・小見出し・統計ラベルの文字色。純黒は使わない。
- **Body Grey — `#999999`**（theme tx1/dk1）: 本文（Lato Light 28pt）の既定文字色。マスターの bodyStyle が tx1 を指す。
- **White — `#FFFFFF`**（bg1/lt1）: 色面上の文字、サーフェス、プログレスバーのトラック。

### Interactive — リンク・ホバー等
スライド本体では未使用だが、テーマに定義済み。Web 展開時はこれを流用する。
- **Link — `#F33B48`**（hlink, 赤）
- **Visited / Hover — `#FFC000`**（folHlink, アンバー）

### Surface & Borders — 背景・ボーダー
- **Page Background — `#FFFFFF`** : 全コンテンツスライドの背景（背景指定 None = 白既定）。
- **Footer / Caption Band — `#F2F2F2`** : 中央下部のページ番号バンド（例 pos 10.417", 13.62", 5.825"×1.38"）。
- **Borders** : 枠線は使用しない（`<a:ln><a:noFill/>`）。区切りは余白と色面で行う。

### Shadow Colors — シャドウ色
**シャドウは一切使用しない。** 全要素フラット。ドロップシャドウ・内側シャドウ・グローのいずれも無し。

### Status — 成功／エラー／警告 / Before/After
元資料に明示的なステータス配色は無い。拡張時の推奨:
- Success → Primary Teal `#00A09C`
- Error / Alert → `#F33B48`（theme hlink を流用）
- Warning → `#FFC000`（theme folHlink を流用）

## 3. Typography Rules

### Font Family
- **見出し（Latin）:** `Montserrat SemiBold`（表紙タイトルのみ `Montserrat` Bold）。フォールバック: `"Montserrat SemiBold", "Montserrat", "Roboto Medium", "Poppins Medium", sans-serif`
- **本文（Latin）:** `Lato Light`。フォールバック: `"Lato Light", "Lato", "Helvetica Neue", Arial, sans-serif`
- **東アジア（EA）:** 見出し `Roboto Medium` / 本文 `Lato Light`（テーマ既定は 游ゴシック Light / 游ゴシック）。和文化する場合は見出し「游ゴシック / Noto Sans JP Medium」、本文「游ゴシック Light / Noto Sans JP Light」。
- テーマ定義フォント（Calibri Light / Calibri）は実スライドでは全て上書きされており、ブランドフォントは Montserrat + Lato。

### Hierarchy
原寸（2×キャンバス）の実測 pt。括弧内は 1× 換算の目安。

| Level | Size (2× / 1×) | Weight | Font | Color | Use |
|---|---|---|---|---|---|
| Cover Title | 130pt / 65pt | Bold | Montserrat | `#FFFFFF` | 表紙の作品名「MAURELLE」 |
| Welcome / Hero | 80pt / 40pt | SemiBold | Montserrat SemiBold | `#FFFFFF` | 大型メッセージ面の見出し |
| Section Heading | 54pt / 27pt | Bold | Montserrat SemiBold | `#494949` | 「OUR CONTENT」等のページ見出し |
| Stat Number | 60pt / 30pt | Bold | Montserrat SemiBold | accent各色 | 「25%」「50%」「70%」 |
| Card / Block Title | ~28–32pt / ~14–16pt | Bold | Montserrat SemiBold | `#494949` | 「Your Title A」「Write Your Title」 |
| Body | 28pt / 14pt | Light | Lato Light | `#999999` | 本文「Most businesses already know…」 |
| Chart Axis Label | 20pt / 10pt | Light | Lato Light | `#999999` | 「High」「Low」 |
| Cover Subtitle | 継承（~24pt 相当）| Regular | Montserrat | `#FFFFFF` | 「SECTION SUBTITLE」 |

### Principles
- 見出しは Montserrat 系で大文字（ALL CAPS）運用が基本（「OUR CONTENT」「GRAPHIC CHARTS」）。
- 本文は Lato Light のみ。本文に Bold を使わず、強調はサイズと色（`#494949`）で行う。
- 見出しと本文で必ず色を分ける（`#494949` 対 `#999999`）。同色で階層を作らない。
- 行間は控えめ（マスター lnSpc 90%）。統計数値は周囲本文の倍以上のサイズで視線を集める。

## 4. Component Stylings

### Cover / Title Slide（スライド0）
- 背景: 全面ベタ塗り **Primary Teal `#00A09C`**（0,0,26.66"×15"）。
- モチーフ: 中央寄りに **Accent2 `#0098A5` の正方形**（pos 8.45", 2.054", 10.893"×10.893"）を重ねる「色面の入れ子」。
- タイトル: 「MAURELLE」`#FFFFFF` / Montserrat Bold / **130pt**（pos 3.298", 6.356"）。
- サブタイトル: 「SECTION SUBTITLE」`#FFFFFF` / Montserrat（タイトル直上 pos 3.431", 5.649"）、字間広めの小キャプション。

### Watermark / Background Title
本テンプレートに薄色ウォーターマーク文字（noFill テキストボックス）は **存在しない**。背景の主張は表紙の色面入れ子と、コンテンツ面の `#F2F2F2` フッターバンドが担う。拡張で巨大英字を置く場合は `#F2F2F2`（背景白との差をごく薄く）を使い、塗りなしテキストボックスに配置すること。

### Section Heading
- 位置: 上部中央寄り（pos 8.053", 1.434", 10.555"×1.01"）。
- スタイル: Montserrat SemiBold / **54pt** / `#494949` / ALL CAPS。

### Card / Block Title + Body（反復ユニット）
- タイトル: Montserrat SemiBold / `#494949`（例「Your Title A」pos w=6.315"）。
- 本文: 直下に Lato Light / **28pt** / `#999999`（例 pos 同x, +0.775"）。
- タイトルと本文は左揃え、間隔 ≈0.0–0.2"。1ブロック幅 6.315"（3カラム時）または 5.524"（カード内）。

### Stat / Number Callout（スライド6）
- 大数字: Montserrat SemiBold / **60pt** / accent色（`#00A09C`「25%」, `#0098A5`「50%」, `#1991AB`「70%」）。
- 添えタイトル: `#494949` Bold、説明本文: `#999999` Lato Light 28pt。

### Icon / Avatar Badge
- バッジ: 一辺 1.345" の accent色正方形（スライド3、`#0098A5` / `#1991AB`）に白文字（「A」「B」「C」）。
- チームアバター: 一辺 1.658" の **正円**（楕円シェイプ）を accent色で塗り、横一列に配置（スライド5）。

### Image Block（スライド4・7）
- 画像/色面は **半キャンバス占有**。例: 写真 pos 6.447", 2.013", **13.765"×7.922"**（スライド7、角丸なし矩形）。
- もう半分にテキスト（見出し `#494949` + 本文 `#999999`）を置く左右2カラム構造。
- 色面版: `#00A09C` の矩形（pos 13.163", 1.411", 12.06"×5.929"）に白文字を載せる。

### Progress Bar（スライド4）
- トラック: `#FFFFFF` 矩形（pos 1.742", 6.172", **4.66"**×0.376"）。
- 充填: `#0098A5` 矩形を同起点に重ねる（幅 **3.303"** ≈ 71%）。角丸なし、高さ 0.376" の細バー。

### Bar / Bubble Chart（スライド6）
- プロット枠: `#FFFFFF` 矩形（pos 1.422", 1.131", 8.057"×5.237"）。
- データ要素: accent色の矩形/正方形をサイズ差で量を表現（`#0098A5` 2.786"×3.061", `#00A09C` 1.951"×2.144", `#1991AB` 4.037"×4.436"）。
- 軸ラベル: 「High」「Low」 Lato Light 20pt `#999999`。

### Buttons / Forms
元資料に無し。拡張時の推奨: 塗り `#00A09C`・文字 `#FFFFFF`・Montserrat SemiBold・**角丸 0（矩形）**・パディング 縦0.3"×横0.6"（1×）。ホバーは `#0098A5` へ。フォーカスリングは `#1991AB` 2px。

## 5. Layout Principles

### Spacing System
- 基準キャンバス 26.66"×15"。外周マージン ≈ 1.4–2.1"（見出し左端 x≈8.05、本文ブロック左端 x≈2.0）。
- カラムギャップ: 3カラム時の列ピッチ ≈ 8.04"（x = 2.135 / 10.172 / 18.21、各幅 6.315"）。
- タイトル↔本文の縦間隔 ≈ 0.77"。

### Grid & Container
- 反復は **3分割グリッド**が基本（3カラム本文 / 3カード / 3統計 / 3アバター）。
- カード幅 5.524"、大カード正方形 5.56"（スライド3）。
- 左右2カラム（半キャンバス）レイアウトを画像スライドで使用。

### Whitespace Philosophy
余白を最大の構成要素として扱う。要素間に枠線を引かず、空白と色面のコントラストで領域を分ける。中央下部の `#F2F2F2` バンド以外、装飾帯は置かない。

### Border Radius Scale
- **角丸は使用しない（0）。** 矩形・正方形は鋭角。
- 唯一の曲線はチームアバターの **正円**（楕円シェイプ、radius = 50%）。

## 6. Depth & Elevation

**完全フラット。シャドウ・エレベーションを使用しない。**

| Level | Shadow | Use |
|---|---|---|
| すべて | none | 全要素が同一平面。前後関係は重なり順と色面のみで表現 |

要素の重なり（表紙の正方形入れ子、プログレスバーのトラック＋充填）は影ではなく**色の差**で奥行きを示す。グロー・反射・ベベルも一切なし。

## 7. Do's and Don'ts

### Do
- 寒色グラデーション群（`#00A09C` / `#0098A5` / `#1991AB` …）を**色面・図形の差し色**として使う。
- 見出しは `#494949` Montserrat SemiBold、本文は `#999999` Lato Light で必ず色を分ける。
- 見出しは ALL CAPS で運用する（OUR CONTENT, GRAPHIC CHARTS）。
- 矩形・正方形は角丸なしの鋭角、アバターのみ正円にする。
- 3分割グリッド（3カラム/3カード/3統計）を反復の基本単位にする。
- 統計数値は 60pt 級の accent色で大きく見せ、視線の起点にする。
- ページ番号は中央下部の `#F2F2F2` バンド内に置く。
- 余白を広く取り、要素間は線ではなく空白で区切る。

### Don't
- 影・グロー・ベベル・枠線を足さない（フラット原則を崩さない）。
- 本文に純黒 `#000000` を使わない（`#999999`、見出しは `#494949`）。
- 本文を Bold にしない（強調はサイズと色で）。
- 角丸を矩形・カードに付けない（半径0を維持）。
- accent色を背景全面に乱用しない（全面塗りは表紙とヒーロー面に限定）。
- Montserrat / Lato 以外の書体を混ぜない（Calibri 等テーマ既定に戻さない）。
- グラデーション塗りを多用しない（本テーマの色面は基本ベタ塗り）。
- 見出しと本文を同色にして階層を潰さない。

## 8. Responsive Behavior

### Breakpoints
- Desktop ≥1024px: 16:9 のフル3カラム/2カラムを維持。
- Tablet 768–1023px: 3カラム → 2カラム、画像+テキストは上下積み。
- Mobile <768px: 全1カラム。見出し ALL CAPS は維持しつつサイズを 1× の 0.7 倍程度に縮小。

### Touch Targets
- バッジ/ボタンは最小 44×44px。アバター正円は最小 64px。

### Collapsing Strategy
- 左右2カラム（画像｜テキスト）はモバイルで画像が上・テキストが下へ縦積み。
- 3統計・3カードは縦スタックし、各統計数値は左、説明文は下に置く。

### Image Behavior
- 画像は角丸なしで `object-fit: cover`、半幅 → モバイルで全幅。
- 色面ブロック（teal矩形）は画像と同じ枠サイズで差し替え可能。

## 9. Agent Prompt Guide

### Quick Color Reference
- `#00A09C` Primary Teal — 表紙背景、第1アクセント、統計「25%」、カード1
- `#0098A5` Accent2 — 入れ子正方形、プログレス充填、統計「50%」、カード2
- `#1991AB` Accent3 — 統計「70%」、カード3、最大バブル
- `#2B85AE` / `#4175A9` / `#5267A5` — グラデーション拡張（中間〜インディゴ終端）
- `#494949` 見出し文字 / `#999999` 本文文字 / `#FFFFFF` 色面上文字・サーフェス
- `#F2F2F2` フッター/ページ番号バンド
- `#F33B48` リンク / `#FFC000` ホバー（拡張）

### Example Component Prompts
1. **表紙:** 「全面 `#00A09C` 背景。中央に一辺が高さの約73%（1280×720なら~390px）の `#0098A5` 正方形（角丸0）を重ね、その上に白の Montserrat Bold 65pt で『MAURELLE』、直上に字間広めの Montserrat 白キャプション『SECTION SUBTITLE』。影なし。」
2. **セクション見出し＋3カラム本文:** 「白背景。上部中央に Montserrat SemiBold 27pt `#494949` の ALL CAPS 見出し。下に等幅3カラム（列ピッチ均等）、各列は `#494949` Bold の小見出し＋直下に Lato Light 14pt `#999999` の本文。区切り線なし。」
3. **3バリューカード:** 「横並びの正方形カード3枚を `#00A09C` / `#0098A5` / `#1991AB` でベタ塗り（角丸0）。各カード内に白の Montserrat SemiBold ラベル。カード下に `#494949` タイトルと `#999999` 本文。影なし。」
4. **統計カラム:** 「Montserrat SemiBold 30pt の大数字を accent色（`#00A09C`/`#0098A5`/`#1991AB`）で表示し、横に `#494949` のタイトルと `#999999` Lato Light 14pt の説明を添える。3件を縦に並べる。」
5. **画像＋テキスト2カラム:** 「左半分に角丸なしのフル画像、右半分に Montserrat SemiBold `#494949` 見出し＋Lato Light `#999999` 本文。または左を `#00A09C` 色面に白文字で差し替え可。」
6. **プログレスバー:** 「高さ約0.19"(1×) の細い `#FFFFFF` トラック矩形に、同起点で `#0098A5` の充填矩形を約71%幅で重ねる。角丸0、影なし。」

### Iteration Guide
1. まず配色を確定: 寒色 accent を「面・図形・統計数字」だけに使い、テキストはグレースケール（`#494949`/`#999999`）に固定する。
2. フォントを Montserrat（見出し・ALL CAPS）＋ Lato Light（本文）に統一。Bold は見出しのみ。
3. 角丸0・影なしのフラット原則を全要素に適用。曲線はアバター正円のみ許可。
4. レイアウトは3分割または左右2カラムの反復で組み、余白を広く取る。
5. pt/インチ値は本書を 2×キャンバス基準として読み、ターゲット解像度に応じて一律スケールする（1280×720 なら 0.5倍）。
6. ページ番号・キャプションは `#F2F2F2` バンドに限定し、それ以外に装飾帯を足さない。
